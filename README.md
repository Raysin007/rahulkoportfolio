Rahul Chettri Portfolio
A modern, responsive personal portfolio website showcasing frontend development skills with dark/light mode toggle, interactive elements, and project highlights.​

Features
Multi-page layout: Home, About, and Projects sections with smooth navigation and animated links.​

Dark/Light Mode: Persistent theme toggle using CSS custom properties and localStorage.​

Interactive Elements: Chuck Norris joke API integration via fetch() on the About page.​

Responsive Design: Bootstrap 5.3.8 with custom CSS for mobile-first layout and hover effects.​

Contact Form: Client-side form with validation placeholders (requires backend for submission).​

Custom Typography: Google Fonts like VALORANT, Tabarra Black, Rogan for unique styling.​

Tech Stack
Category	Technologies
HTML/CSS	HTML5, CSS3, Bootstrap 5.3.8​
JavaScript	Vanilla JS, Fetch API, localStorage​
External	Font Awesome icons, CDN fonts​
Assets	Resume PDF download, images (red.jpg, craftylogo.png, etc.)​
Pages
index.html: Hero section with resume download, social links (GitHub, LinkedIn, Instagram), and design quote.​

about.html: Developer bio, Chuck Norris jokes, contact form.​

projects.html: Cards for Crafty Vision (e-commerce), Summit Himalayan (homestay site), and this portfolio.​

Quick Start
Clone the repo: git clone https://github.com/Raysin007/rahul-portfolio.git

Open index.html in a browser – no build tools needed.

Toggle dark mode and test jokes/form on About page.​

Deployment
Hosted on Netlify: [rahulkoportfolio.netlify.app]. Uses relative paths for easy static hosting.​

Assets Note
Images (red.jpg, bean.gif, etc.) and atsresume.pdf must be in the root directory.
